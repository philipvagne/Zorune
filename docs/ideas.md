## Workspace-Scoped Overlay System

### Status

Future investigation.

### Idea

Examples:

* Create Team
* Add Member
* Edit Team
* Delete Team
* Create Project
* Edit Project

### Reason

Current popups work, but they feel generic. A workspace-scoped overlay could feel more integrated with Zorune.
